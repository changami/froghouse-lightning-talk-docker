Theme
---
docker-composeでWordPress開発環境構築を5分で作って捨てる

Usage
---
Initialize:
```sh
git clone https://github.com/changami/froghouse-lightning-talk-docker.git

cd froghouse-lightning-talk-docker
./init.sh
```

Run the environment:
```sh
docker-compose up -d
```

Down the environment:
```sh
docker-compose down
```
